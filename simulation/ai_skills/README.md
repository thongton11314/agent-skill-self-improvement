# AI-Generated Skills

This directory stores skill artifacts generated autonomously by SkillForge.

Skills are generated by running:

```python
from skillforge import SkillForge

forge = SkillForge.from_config("config.yaml")

# Load simulation tasks
import json
with open("simulation/tasks/simulation_tasks.json") as f:
    tasks = json.load(f)

# Generate skills for each task
for task in tasks:
    skill = forge.evolve_skill(task, max_evolution_rounds=5)
    forge.skill_bank.store(skill)
    
    # Export skill artifacts
    for name, content in skill.artifacts.items():
        path = f"simulation/ai_skills/{task['task_id']}_{name}"
        with open(path, "w") as f:
            f.write(content)
```

After generation, each task will have corresponding files here (e.g., `SIM-001_SKILL.md`).
